# Naples SLR + Tide Flooding Viewer

This package is a GitHub Pages–ready interactive viewer for sea-level-rise and tide inundation in coastal Naples, Florida.

## Features
- Satellite basemap (Esri default, Google satellite/hybrid optional)
- Decade selector: 2050 or 2100
- Scenario selector: SSP2-4.5 or SSP5-8.5
- Interactive SLR and tide sliders
- Flood depth hover tooltip
- About/info modal explaining methodology, scenarios, and limitations
- Flood opacity control

## Notes
- The viewer uses a bathtub inundation approach with a coastal-connectivity constraint.
- Internet access is required for the satellite basemap tiles to load.
